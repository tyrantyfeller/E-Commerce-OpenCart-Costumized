# Projeto Draculartes 
___
## 💻 Tecnologias:

 - php 7.4
 - mysql 5.7
 - docker
 - Opencart CMS

___
## 🐋 Como rodar o projeto
___

##### Ambiente local

Iniciando serviço do docker

```shell
sudo service docker start
```

levantando imagem docker

```shell
docker-compose up -d
```

executando linha de comando da imagem docker

```shell
docker exec -it php-app sh
```
